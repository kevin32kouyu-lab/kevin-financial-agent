# ROSE5780 Final Report Source Notes

These notes record the verified sources used for the literature survey. The report should cite them as supporting references, not claim that this project is a direct copy of any one system.

## BloombergGPT

- Title: BloombergGPT: A Large Language Model for Finance
- Year: 2023
- URL: https://arxiv.org/abs/2303.17564
- Relevance: Shows the value of domain-specific financial language models trained and evaluated for finance tasks. It supports the report's argument that finance needs specialized language and data handling rather than a generic chatbot only.

## FinGPT

- Title: FinGPT: Open-Source Financial Large Language Models
- Year: 2023
- URL: https://arxiv.org/abs/2306.06031
- GitHub: https://github.com/AI4Finance-Foundation/FinGPT
- Relevance: Provides an open-source financial LLM direction and emphasizes data-centric financial AI. It is useful for comparing this project with broader financial LLM infrastructure.

## FinRobot

- Title: FinRobot: An Open-Source AI Agent Platform for Financial Applications using Large Language Models
- Year: 2024
- URL: https://arxiv.org/abs/2405.14767
- GitHub: https://github.com/AI4Finance-Foundation/FinRobot
- Relevance: A direct reference for financial LLM agents and agent toolchains. It supports the methodology section on specialized agents for financial analysis.

## TradingAgents

- Title: TradingAgents: Multi-Agents LLM Financial Trading Framework
- Year: 2024
- URL: https://arxiv.org/abs/2412.20138
- GitHub: https://github.com/TauricResearch/TradingAgents
- Relevance: Shows a multi-agent trading architecture with different analyst and researcher roles, including bull and bear reasoning. This is related to the project's Bull, Bear, and Arbiter design, while this project focuses more on explainable research reports than automated trading.

## FinMem

- Title: FinMem: A Performance-Enhanced LLM Trading Agent with Layered Memory and Character Design
- Year: 2023
- URL: https://arxiv.org/abs/2311.13743
- Relevance: Shows how memory can help financial agents use historical context and structured experience. It supports this project's lightweight long-term preference and history design.

## FinRL

- Title: FinRL: A Deep Reinforcement Learning Library for Automated Stock Trading in Quantitative Finance
- Year: 2020
- URL: https://arxiv.org/abs/2011.09607
- GitHub: https://github.com/AI4Finance-Foundation/FinRL
- Relevance: Provides a traditional quantitative finance reference for trading environments, transaction cost, risk constraints, and backtesting. It helps position this project as a GenAI research/reporting agent rather than a pure trading strategy engine.
