# 真实回测、账户记忆、生产治理与自动化验证开发计划

## Summary
本轮把项目从“可演示”推进到“可上线前验证”：回测口径更真实，长期记忆升级到账户体系，补齐 CI、健康检查、权限、审计、数据刷新任务和浏览器级端到端测试。

## Key Changes
- 回测 V2：支持交易成本、滑点、分红模式、简化税费和再平衡频率。
- 账户记忆：新增本地账号、会话、登录状态和浏览器记忆绑定到账户。
- 生产治理：新增 `/readyz`、GitHub Actions、操作审计和可选监控配置。
- 数据更新：新增股票池、宏观和全量刷新接口，并记录刷新任务历史。
- 前端测试：新增 Playwright 终端四页 smoke test。

## Test Plan
- 后端：账户注册/登录、审计、记忆绑定、回测 V2、数据刷新任务。
- 前端：`npm run build` 和 `npm run test:e2e`。
- 部署：Railway 后检查 `/healthz`、`/readyz`、四个终端路由。
